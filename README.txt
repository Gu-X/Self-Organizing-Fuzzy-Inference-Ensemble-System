# Self-Organizing-Fuzzy-Inference-Ensemble-System

This work is described in:
===============================================================================================================================
Gu, X., Angelov, P. and Zhao, Z.(2021). Self-Organizing Fuzzy Inference Ensemble System for Big Streaming Data Classification. 
Knowledge-Based Systems, accepted for publication.
